# BookWise – Book Recommendation App

##  Overview
BookWise is a simple React-based web application that helps users discover books based on their interests. Users can search for books or explore recommendations by category.

##  Features
- Search books by title or keyword
- View book details (title, author, cover image)
- Category-based recommendations (Fiction, Self-help, Mystery, etc.)
- Clean and simple UI

##  Tech Stack
- React.js
- HTML
- CSS
- Google Books API

##  API Used
https://www.googleapis.com/books/v1/volumes?q=

##  Objective
To build a beginner-friendly book recommendation system using React and API integration.
